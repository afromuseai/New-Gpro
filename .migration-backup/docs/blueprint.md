# **App Name**: GrowthPulse Pro

## Core Features:

- User Authentication & Profile: Secure user registration, login, and profile management for account settings.
- Social Account Connection: Integrate and authorize social media accounts (Instagram, TikTok, Twitter, LinkedIn) via OAuth for data access.
- Content Scheduling & Calendar: Schedule social media posts across connected platforms with a unified calendar view.
- Basic Analytics Dashboard: View fundamental growth metrics (follower count, engagement rates) and post performance across platforms.
- AI Bot Profile Tool: Generate realistic bot profile data, including names, bios, avatars (AI-generated or placeholder), and platform-specific attributes, for testing purposes.
- Subscription Management: Manage user subscriptions, billing information, and access to premium features.

## Style Guidelines:

- Dark color scheme, reflecting a modern, data-centric dashboard environment. Primary color: deep indigo (#4A148C) to convey reliability and innovation. Background color: very dark, desaturated indigo (#1A121F) for emphasis on data visualizations. Accent color: vibrant magenta (#D81B60) for calls to action and important alerts, providing strong contrast.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, techy, and direct feel. Body text: 'Inter' (sans-serif) for excellent readability across all data displays and descriptive content.
- Minimalist line-art icons to maintain a clean aesthetic and ensure clarity of navigation and data points.
- A responsive, single-page application layout with a fixed sidebar for primary navigation and a clear main content area for dashboard views and data presentation.
- Subtle, fast animations for state changes, chart loading, and modal transitions to enhance perceived responsiveness without being distracting.
